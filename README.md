## English

WeiClaw is an intelligent hardware device that connects to your Agent-equipped computer. By monitoring the Agent status in real time and intelligently taking over the message channel, WeiClaw allows your computer to sleep when idle and wake up automatically on demand.

### Key Features
*   **Smart On-Demand Work:** Agents activate only when needed and sleep when idle, significantly reducing energy consumption.
*   **WeChat Account Sharing:** Share your ClawBot directly without needing to register new accounts.
*   **Automatic Message Wake-up:** Seamlessly wake up your computer and agent through message interaction.
*   **Strong Compatibility:** Fully supports OpenAI-compatible interfaces (e.g., OpenClaw, Hermes Agent).

### Usage scenario
Gain control over the Agent anytime, anywhere. It can be flexibly deployed on both home and office computers, and can be remotely woken up via messages as needed, ensuring a seamless and smooth response.

## 中文

WeiClaw 是连接 Agent 与电脑的智能硬件。通过实时监测 Agent 状态、智能接管消息通道，它让电脑无需长期高负载运行 —— 闲置时自动睡眠省电护机，有任务时即刻唤醒响应，实现高效节能的按需工作模式。

### 产品特性
*   **智能按需工作，节能降耗：** 仅在需要时启动 Agent，闲置时睡眠，大幅降低能耗，有效延长电脑使用寿命。
*   **微信账号共享，无需额外申请：** 无需新注册微信账号，直接共享使用 ClawBot，WeiClaw 实时监控 Agent，智能接管消息通道。
*   **全自动消息唤醒，使用无缝感知：** 无需手动切换、无需关注 Agent 工作状态，WeiClaw 自动介入，全程无缝丝滑响应。
*   **超强兼容，适配主流智能代理：** 全面支持 OpenAI 兼容接口，OpenClaw、Hermes Agent 等均可无缝体验。

### 使用场景
随时随地掌控 Agent，家用电脑、办公电脑均可灵活部署，按需远程消息唤醒、无缝丝滑响应。
